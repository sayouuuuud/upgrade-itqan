# خطة الاختبار الشامل لمنصة الأكاديمية

## ملخص المشروع
تم بنجاح توسيع منصة إتقان لتشمل أكاديمية قرآنية متكاملة مع أدوار متعددة (طالب، مدرس، أدمن، ولي أمر)، ودرجات متقدمة، ونظام نقاط وشارات، وغيرها.

## المراحل المكتملة

### المرحلة 1: قاعدة البيانات ✓
- تم إنشاء 17 جدول جديد لدعم الأكاديمية
- الجداول تشمل: courses, lessons, enrollments, tasks, memorization_log, points, badges, competitions, forum, fiqh_questions, sessions

### المرحلة 2: الأنواع والمصادقة ✓
- تحديث `types.ts` بأدوار وأنواع جديدة
- تحديث `auth.ts` بدوال `hasAcademyRole` و `getAcademyRole`
- ملف `lib/types/academy.ts` مع 455 سطر من الأنواع المحددة

### المرحلة 3: Middleware والمكونات ✓
- تحديث middleware.ts لدعم مسارات الأكاديمية
- إنشاء `mode-switcher.tsx` للتبديل بين الأوضاع
- إنشاء `academy-dashboard-shell.tsx` مع sidebar ديناميكي حسب الدور

### المرحلة 4: صفحات الطالب ✓
- 10 صفحات للطالب: dashboard, courses, memorization, tasks, sessions, progress, path, leaderboard, badges
- جميع الصفحات متصلة بـ API وتظهر بيانات ديناميكية

### المرحلة 5: صفحات المدرس ✓
- 6 صفحات للمدرس: dashboard, courses, schedule, tasks, students, live
- واجهات لإدارة الدورات والجلسات والمهام

### المرحلة 6: صفحات الأدمن ✓
- 9 صفحات للأدمن: dashboard, courses, teachers, paths, competitions, forum, fiqh, leaderboard, invitations
- لوحة تحكم شاملة لإدارة جميع جوانب الأكاديمية

### المرحلة 7: API Routes ✓
- 20+ endpoint API لجميع العمليات
- API للطالب (enrollments, memorization, points, badges)
- API للمدرس (courses, sessions, tasks, students, live)
- API للأدمن (stats, courses, teachers, paths, competitions, forum, fiqh, leaderboard, invitations)

### المرحلة 8: نظام النقاط ✓
- `lib/academy/points.ts` مع 250 سطر
- دوال لمنح النقاط الآلية (memorization, tasks, attendance, competitions, etc.)
- نظام الشارات التلقائي
- حساب الانجازات والعصات

### المرحلة 9: الدروس العامة والدعوات ✓
- صفحات عامة للدروس والدعوات
- نظام الاشتراك في الدروس العامة
- نظام الدعوات مع رموز صلاحية

## الميزات المطبقة

### للطالب
- عرض الدورات المسجلة والتقدم
- تسجيل الحفظ مع تقييم الجودة
- عرض المهام والجلسات المباشرة
- نظام النقاط والشارات
- لوحة المتصدرين
- مسارات التعلم المخصصة

### للمدرس
- إنشاء وإدارة الدورات
- جدولة الجلسات المباشرة
- إنشاء المهام وتتبع الإرسالات
- عرض قائمة الطلاب والإحصائيات
- بث مباشر مع إدارة المشاركين

### للأدمن
- إحصائيات شاملة
- إدارة الدورات والمدرسين
- إدارة مسارات التعلم
- إدارة المسابقات
- إدارة المنتدى والأسئلة الفقهية
- لوحة المتصدرين
- إدارة الدعوات

### نظام الألعاب
- منح النقاط تلقائياً لـ:
  - إكمال المهام (50 نقطة)
  - الحفظ الممتاز (100 نقطة)
  - الحضور (20 نقطة)
  - الفوز في المسابقات (500 نقطة)
  - الإجابة في المنتدى (25 نقطة)
- الشارات التلقائية (first_steps, rising_star, champion, legend, master)
- مكافآت العصات (3 أيام، 7 أيام، 30 يوم)
