## ✅ نمط الترجمة الموحد

كل ملف يتبع **نفس الخطوات:**

```typescript
// 1️⃣ إضافة context
const { t } = useI18n()
const [namespace] = (t as any).[categoryName]

// 2️⃣ استبدال الترجمات
"النص العربي" → {namespace?.key ?? 'English fallback'}

// 3️⃣ إضافة namespace في ar.ts + en.ts
[categoryName]: {
  key: 'النص العربي',
}

// 4️⃣ TypeScript check
pnpm exec tsc --noEmit

// 5️⃣ Commit
git add -A && git commit -m "feat(i18n): translate [filename]"

// 6️⃣ Push
git push origin v0/m01061195172-5942-4c4c8509
```
